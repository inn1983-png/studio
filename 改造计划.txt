# 6. 资产库改造

资产库是你超过普通 ai movie studio 的关键。

## 6.1 角色库

```json
[
  {
    "id": "char_001",
    "name": "张横",
    "gender": "男",
    "age": "三十五岁",
    "identity": "宋代捕快",
```

```text
同一角色只允许一个 ID
性别一旦确认不得变化
角色名必须来自角色库
不在角色库的人统一叫“龙套”
支持人工编辑
支持替换参考图
```

# 17. 我建议的最终执行优先级

让 Codex 自己选择细节可以，但大方向别让它乱跑。你只管盯住这条主线：

```text
先拿回 API 控制权
再建立 Txtovideo 数据结构
再接资产库和分镜 JSON
再接 ComfyUI
再接视频
最后才改前端和打包
```

品牌替换建议放在两个阶段：

```text
早期：改配置、标题、LOGO、关于页，避免一直用 AICON 名字开发
后期：改安装包、桌面图标、自动更新、发布包名称
```

不要一开始就把所有 AICON 字符串暴力替换。
正确做法是：

```text
先加 brand.json
再让前端和后端读取 brand 配置
最后逐步清理硬编码品牌名
```

这样最稳。
